## Day 10 capstone
The project for today was building a simple calculator. It will allow a user to choose a number, an operator, and another number, then produce the output. it will then allow the user to either use that as the input for the next calculation, start over, or quit

## Flow chart:

1. Accept first num input
2. Accept operator input
3. Accept second num input
4. Display output
5. Ask whether user  would like to operate on the result
6. If yes, assign value to num1 and go to step 2
7. else go to step 1
