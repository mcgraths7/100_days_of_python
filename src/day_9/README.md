## Day 9 capstone
The project for today was building a silent auction. The user is prompted for their name and their bid and whether there are any other bidders. It will then stop and reveal the winner when no more bidders are present.

## Above and Beyond
Converted from First-Price sealed bid style to Second-Price sealed bid style. Also generated some ridiculous auction items that are randomly chosen each auction

## Flow chart:

1. Get user name and bid
2. prompt for additional bidders
3. clear screen
4. repeat until 2 == "no"
5. Print winner and bid price following Second-price rules
