# Day 11

Today was all about bringing it all together:

## Day 11 capstone
The project for today was building a blackjack game

## Flow chart:
1. Get dealt a hand of two cards (only values). You can see the dealers first card, you can see your and other players cards
2. Players take turns hitting until they stay or bust
3. If over 21, bust
4. Dealer will hit until soft 17 is hit

## Above and beyond
1. For this one I have built a "fair" deck, where there are only 52 cards, and if you keep playing over and over again, eventually you will have to reshuffle a new deck into the current one.
2. attempt functional programming
2. Add betting
3. Log the players balance to a file so the most recent balance can be read when a new game starts (persists through exiting)
